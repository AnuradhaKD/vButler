info Overview

"Every Browns guest deserves a personal butler — available 24/7, across every property, in the palm of their hand."

Destinity vButler is the official guest-facing digital companion for Browns Hotels & Resorts. Version 1.0 delivers a complete, production-ready Progressive Web App covering every major guest journey touchpoint — from the moment a reservation is confirmed, through check-in, the in-stay experience, and post-checkout.

The app is built on a reservation-centric model: every feature — services, billing, complaints, dining, wellness — is contextual to the guest's active reservation. A single login gives access to all reservations across all Browns properties, past and present.

info Not in v1 (Planned)

  • · Live PMS integration (all data is mock JSON in v1)
  • · Real payment gateway (payment UI only — no actual transaction processing)
  • · WhatsApp Business API dispatch (UI flow built; API wiring pending)
  • · Push notifications (in-app notifications fully built; browser push pending)
  • · Multi-language support (UI is English only in v1; translations deferred)
  • · In-app guest-to-staff chat
  • · New reservation booking engine (Destinity vButler manages existing reservations only)
